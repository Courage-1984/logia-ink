# Image Generation Prompt Guide for Logi-Ink Website

This guide provides comprehensive information for generating images using AI image generators (like Gemini, DALL-E, Midjourney, etc.) for the Logi-Ink website. Use this document to create prompts that match the site's cyberpunk aesthetic and brand identity.

- **Mood**: Innovative, cutting-edge, professional, modern

### Key Visual Elements

## üé® Color Palette

### Primary Background Colors
- **Deep Black**: `#0a0a0a` - Main background color
- **Dark Blue-Black**: `#1a1a2e` - Secondary backgrounds, cards, sections
- **Navy Blue**: `#16213e` - Tertiary backgrounds

### Accent Colors (Use prominently in images)
- **Electric Cyan**: `#00ffff` - Primary accent (use most frequently)
- **Hot Magenta**: `#ff00ff` - Secondary accent
- **Electric Green**: `#00ff00` - Success/positive states
- **Electric Blue**: `#0066ff` - Info states
- **Hot Pink**: `#ff0080` - Error/warning states

### Text Colors
- **White**: `#ffffff` - Primary text
- **Light Gray**: `#b0b0b0` - Secondary text
- **Gray**: `#666666` - Muted text

### Color Usage Guidelines
- **Primary colors** (cyan and magenta) should be used together or separately for accent
- Use **green, blue, and pink** sparingly for variety
- Always maintain **high contrast** between dark backgrounds and bright accents
- Include **glow effects** around colored elements
- Use **gradients** between accent colors (cyan-to-magenta, etc.)

**File Format**: WebP, AVIF

## üé® Prompt Template Structure

Use this template structure for consistent results:

```
[Subject/Concept], [Background Color (#hex)], [Primary Accent Color (#hex)], [Secondary Accent Color (#hex)], [Key Visual Elements], [Style/Aesthetic], [Dimensions], [Additional Requirements]
```

### Template Breakdown:
1. **Subject/Concept**: What the image represents (abstract, service, project, etc.)
2. **Background Color**: Use exact hex codes (#0a0a0a, #1a1a2e, #16213e)
3. **Primary Accent**: Cyan (#00ffff) or Magenta (#ff00ff)
4. **Secondary Accent**: Complementary color from palette
5. **Key Visual Elements**: Grids, particles, networks, geometric shapes, etc.
6. **Style/Aesthetic**: Cyberpunk, futuristic, tech, digital, etc.
7. **Dimensions**: 1920x1080, 1024x1024, etc.
8. **Additional Requirements**: Glowing effects, transparency, depth, etc.

## üé® Technical Specifications

### Image Dimensions
- **Hero Backgrounds**: 1920x1080 (16:9)
- **Banners**: 1920x1080 or 1920x720
- **Parallax**: 1920x1080 or 1920x1200 (taller for parallax effect)
- **Icons**: 512x512 or 1024x1024 (square)
- **Abstract Backgrounds**: 1920x1080 or custom

### File Formats
- **Primary**: WebP (best compression)
- **Alternative**: AVIF (better compression, newer format)
- **Icons**: PNG (transparent) or SVG (vector)
- **Fallback**: JPG (if needed)

### Responsive Sizes
Generate multiple sizes for responsive images:
- **480w**: Mobile (480px width)
- **768w**: Tablet (768px width)
- **1024w**: Desktop small (1024px width)
- **1280w**: Desktop large (1280px width)

### Color Accuracy
- Always use exact hex codes in prompts
- Verify colors match brand palette
- Maintain high contrast ratios
- Test readability with text overlays

## üé® Common Variations

### Color Variations
- **Cyan Primary**: Use cyan (#00ffff) as main accent, magenta (#ff00ff) as secondary
- **Magenta Primary**: Use magenta (#ff00ff) as main accent, cyan (#00ffff) as secondary
- **Multi-Color**: Combine cyan, magenta, green, blue, pink for variety
- **Monochromatic**: Single accent color with varying intensities

### Intensity Variations
- **Subtle**: Lower opacity, softer glows, minimal elements
- **Moderate**: Balanced contrast, medium glow effects
- **Bold**: High contrast, strong glows, prominent elements

### Complexity Variations
- **Minimalist**: Simple patterns, few elements, clean
- **Moderate**: Balanced composition, multiple elements
- **Complex**: Dense patterns, many elements, layered

## üé® Troubleshooting

### If Images Are Too Bright
- Emphasize "deep black background (#0a0a0a)"
- Add "dark atmosphere" to prompt
- Specify "high contrast with dark base"
- Mention "moody and dark"

### If Colors Don't Match
- Use exact hex codes in prompts
- Specify "electric cyan (#00ffff)" not just "cyan"
- Mention "neon" or "glowing" for intensity
- Verify color accuracy in image editor

### If Images Are Too Busy
- Add "minimalist" or "subtle" to prompt
- Reduce number of elements mentioned
- Focus on single concept
- Mention "clean composition"

### If Images Lack Glow Effects
- Explicitly mention "glowing effects"
- Add "neon glow" to prompt
- Specify "luminous" or "radiant"
- Mention "halo effect" or "light emission"
